# 🏛️ Obafemi Awolowo University (OAU) - Official Web Portal

A high-performance, modern web application for **Obafemi Awolowo University, Ile-Ife**. This project serves as a comprehensive portal for university services, featuring a dedicated **Student Health Centre** management system.

Built with "Great Ife" excellence in mind.

## 🚀 Live Demo
**View the site here:** [https://healthcentre_oau.vercel.app](https://healthcentre_oau.vercel.app)

---

## ✨ Features

### 🏥 Student Health Centre Portal
- **24/7 Emergency Response:** Immediate access to campus ambulance contacts.
- **Appointment Booking:** Digital scheduling for General Consultation, Dental, and Lab tests.
- **Service Directory:** Overview of all medical departments including Pharmacy and Optometry.
- **Mobile Responsive:** Optimized for students to access care quickly from their smartphones.

### 🏫 University Landing Page
- **Modern UI/UX:** Clean, professional design using OAU’s official Midnight Blue and Gold palette.
- **Academic Overview:** Quick access to the 13 faculties and research centers.
- **Fast Loading:** Built with Next.js for near-instant page transitions.

---

## 🛠️ Tech Stack

- **Framework:** [Next.js 14](https://nextjs.org/) (App Router)
- **Language:** [TypeScript](https://www.typescriptlang.org/) (Strictly typed for reliability)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Deployment:** [Vercel](https://vercel.com/)
